## Как запустить проект

1. Скачать и установить [Ren'Py SDK](https://www.renpy.org/latest.html) (рекомендуется последняя стабильная версия).
2. Запустить Ren'Py Launcher.
3. Выберать **"Choose Project / Add Project"** и указать папку с этим репозиторием.
4. Выберать проект в списке и нажать **"Launch Project"**.

Проект готов к запуску без дополнительных зависимостей (чистый Ren'Py).
